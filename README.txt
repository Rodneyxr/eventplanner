Installation Instructions
-----------------------------------------------------
Option 1: Move app/eventplanner.apk onto your android device and install the apk file.
Option 2: Download and install the apk from https://rodneyxr.com/ep/eventplanner.apk

Get the source code
-----------------------------------------------------
Open Android Studio and select open existing project. Point it to this directory 
and Android Studio will open the project.
